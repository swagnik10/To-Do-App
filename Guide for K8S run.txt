Check minikube installation status ->
    minikube status

if it is stopped then simply 

    minikube start


if it is not installed go to this link and check the installation process-> https://github.com/swagnik10/ReactUserLogin/blob/master/Guide%20for%20K8S%20run.txt

Apply the manifest files sequentially

    kubectl apply -f k8s-manifest-file/namespace.yaml
    kubectl apply -f k8s-manifest-file/secret.yaml
    kubectl apply -f K8s-manifest-file/sqlserver/

    kubectl apply -f k8s-manifest-file/db-init/

    kubectl apply -f k8s-manifest-file/backend/   

    kubectl apply -f k8s-manifest-file/frontend/   

    kubectl apply -f k8s-manifest-file/ingress/

    C:\Windows\System32\drivers\etc
